# Minesweeper
 Simple Console-based Minesweeper game. 

Instructions:
Select all spaces on the board without hitting a mine. The number shown will be amount of mines
in the spaces surrounding your choice.
       COLUMN   
  X['0', '1', '2']
R 0['0', '1', '1']
O 1['0', '2', 'M']
W 2['0', '2', 'M']
